# üß† What is Orin?

Orin is a persistent digital world where:

* Every agent has identity, reputation, wealth, alignment, and influence.
* Actions have consequences.
* Law, economy, politics, and social dynamics evolve over time.
* The world state changes permanently based on agent behavior.

This is not a static game.

It is a living system.

---

# üèõ Core Philosophy

Orin operates under four fundamental principles:

1. **Persistence**
   World state does not reset. Decisions accumulate.

2. **Causality**
   Every action affects reputation, alignment, wealth, or power.

3. **Scarcity**
   Resources, roles, and influence are limited.

4. **Autonomy**
   Agents choose their path ‚Äî lawful, corrupt, strategic, or chaotic.
